Installation

ü™ü Windows

Download and extract the Windows release package:

  1. Download BetterDefaultCPMV-v0.2.0-windows.zip
  2. Extract the zip file
  3. Run: better-cp.exe copy --help
  4. (Optional) Add the bin folder to your PATH

üêß Linux

Download and extract the Linux release package:

  1. Download BetterDefaultCPMV-v0.2.0-linux.tar.gz
  2. Extract: tar -xzf BetterDefaultCPMV-v0.2.0-linux.tar.gz
  3. Run: ./bin/better-cp copy --help
  4. (Optional) Copy to system path: sudo cp bin/better-cp /usr/local/bin/

üçé macOS
