Production Architecture

Built with Rust for safety, performance, and reliability.

Core Design Principles

  • Memory Safe: 100% safe Rust ‚Äì no unsafe code blocks
  • Concurrent: Async/await with tokio for non-blocking operations
  • Atomic: Temp-then-rename pattern prevents partial overwrites
  • Efficient: 64MB chunked I/O optimized for all storage types
  • Observable: Real-time metrics and detailed error messages
  • Testable: Comprehensive test suite with 50%+ coverage

Module Architecture

Transfer Engine

copy.rs ‚Äì Async file/directory copy with chunked I/O and proper error handling

Safety Layer

prompt.rs ‚Äì User interaction and confirmation handling

Verification

verify.rs ‚Äì SHA-256 checksum computation and validation

Resume State

resume.rs ‚Äì Checkpoint persistence and recovery logic

Progress Tracking

progress.rs ‚Äì Real-time metrics and ETA calculation

Output Control

output.rs + json_output.rs ‚Äì Verbosity management and JSON serialization

Error Handling

error.rs ‚Äì Custom error types with recovery suggestions

Configuration

config.rs ‚Äì TOML config file parsing and XDG support

CLI Layer

cli.rs ‚Äì Clap-based argument parsing with 20+ flags
